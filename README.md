# 📚 AI Study Planner

A Python tool that creates personalized study schedules
based on your subjects and exam date.

## 💡 Why I Built This

Every student struggles with organizing study time.
I built this tool to solve that problem using Python,
automatically calculating how to divide study time
across subjects before an exam.

## 🚀 How It Works

1. Enter your name and subjects
2. Enter your exam date
3. AI calculates days available
4. Generates a personalized day-by-day study plan

## 🛠️ Built With

- Python
- Date and time calculations
- Logic-based schedule generation
